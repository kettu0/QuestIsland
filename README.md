# Quest Island
First web project for coding school, Codecool. The platform on which Codecool students register and get experience points, codecool coins and artifacts for completing additional school tasks. Their progress is verified by the mentors. The platform has a cheerful, colorful design with pirate adventure motives. 

# Technological stack
C#, ASP.NET Core, MVC, DAO, SQL, PostgreSQL, JavaScript, GIT, HTML5, CSS3
